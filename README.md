# house-price-prediction
House Price Prediction is a Machine Learning-based web application developed to estimate residential property prices using important property features such as State, City, Property Type, Bedrooms (BHK), and Property Size. The project is built using Python, Flask, Scikit-learn, Pandas, NumPy, HTML, CSS, and JavaScript to provide a simple and interactive user experience.

The application follows a complete machine learning workflow, including data preprocessing, handling missing values, feature encoding, model training, evaluation, and deployment. A Linear Regression model is trained on an Indian housing dataset to learn the relationship between property characteristics and house prices. The trained model is then integrated into a Flask web application, allowing users to enter property details and receive an estimated house price instantly.
